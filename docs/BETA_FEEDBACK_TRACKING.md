## üé§ User Interview Questions

### Semi-Structured Interview Guide (45 minutes)

**Section 1: Background (5 min)**

1. Tell me about your church and quiz program
   - Size of congregation
   - How long running quiz program
   - Frequency of quizzes

2. What were you using before Smart eQuiz?
   - Paper-based / Other digital tool / Nothing formal
   - What were the pain points?

**Section 2: Onboarding Experience (10 min)**

3. Walk me through your first time using Smart eQuiz
   - What was your initial reaction?
   - Did anything surprise you (good or bad)?

4. How easy was it to create your first tournament?
   - Scale 1-10
   - What made it easy/hard?

5. Did you use the Quick Start Guide?
   - If yes: Was it helpful?
   - If no: Why not?

6. What questions did you have that weren't answered?

**Section 3: Feature Deep-Dive (15 min)**

7. Which features are you using regularly?
   - Probe: Tournament creation, question generation, live mode, etc.

8. Which features haven't you tried yet?
   - Why not?
   - Are they not relevant or not discovered?

9. Tell me about using the AI question generator
   - How often do you use it?
   - Quality of generated questions?
   - Any issues?

10. Have you run a live moderated tournament?
    - If yes: How did it go?
    - If no: What's preventing you?

11. What about the analytics/reports feature?
    - Do you use it?
    - What insights have you gained?

**Section 4: Participant Feedback (10 min)**

12. How do your participants feel about the platform?
    - Direct quotes if possible

13. Have you noticed any changes in engagement?
    - Completion rates
    - Enthusiasm
    - Learning outcomes

14. Any complaints from participants?
    - Technical issues
    - Confusion
    - Feature requests

**Section 5: Future Vision (5 min)**

15. If you could add ONE feature, what would it be?
    - Why is this important?

16. What would make Smart eQuiz a "must-have" vs "nice-to-have"?

17. Would you pay for this platform?
    - If yes: How much feels fair?
    - If no: What's missing?

18. Would you recommend Smart eQuiz to other churches?
    - Why or why not?
    - What would you tell them?

**Wrap-Up:**
- Any other feedback?
- Can we follow up in a month?
- Would you be willing to be a reference customer?

# Project Summary
The Smart eQuiz Tournament Platform is a robust web application aimed at facilitating Bible quiz tournaments for churches and educational institutions. It offers a rich experience for participants, spectators, and administrators through features like multi-role authentication, advanced tournament management, AI-generated questions, and comprehensive analytics tools. The platform's payment management system has been enhanced to support local payment gateways, ensuring improved accessibility and security for users.

# Project Module Description
- **Dashboard**: Central hub for user statistics, active tournaments, and navigation, now includes security logs and notifications.
- **Admin Sidebar**: Navigation for user management, analytics, branding, and system settings.
- **Question Bank**: Manages quiz questions with AI-assisted generation and filtering, fully functional with CRUD operations.
- **Practice Mode**: Allows users to practice quizzes with customizable settings and progress tracking.
- **Tournament Builder**: Wizard for creating and managing tournaments.
- **Tournament Engine**: Interface for professional tournament management with detailed status control.
- **Live Match**: Facilitates real-time competitions with leaderboards and participant tracking.
- **AI Question Generator**: Generates Bible quiz questions using AI.
- **Tenant Management**: Manages organizations and their subscriptions.
- **System Settings**: Configures global platform settings and features.
- **Branding Settings**: Customize visuals to match organizational branding.
- **Authentication System**: Manages user login, registration, and session persistence.
- **Payment Management**: Enhanced to include various local payment gateways with advanced security features.
